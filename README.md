# 项目简介

手写数字体识别、猫狗分类、自动写诗及情感分析项目，融合了计算机视觉与自然语言处理技术。项目主要包含手写数字图像识别、猫狗图片自动分类、基于深度学习的自动诗歌创作以及情感分析功能模块。技术特点包括采用卷积神经网络（CNN）进行图像识别，循环神经网络（RNN）实现文本生成，以及深度学习算法进行情感分类。项目提供完整的源代码及文档说明，便于用户理解与应用。


## 核心功能
- 手写数字图像识别：使用卷积神经网络(CNN)技术识别手写数字图像
- 猫狗图片自动分类：实现动物图片的智能分类功能
- 自动诗歌创作：基于深度学习的循环神经网络(RNN)生成诗歌
- 情感分析：运用深度学习算法进行文本情感分类

## 适用场景
- 教育领域：作为机器学习和深度学习的教学案例
- 研究项目：计算机视觉和自然语言处理技术的研究参考
- 开发实践：深度学习模型的应用与二次开发
- 学术论文：相关算法的实现与验证

## 技术特点
- 采用卷积神经网络(CNN)架构进行图像识别任务
- 利用循环神经网络(RNN)实现文本生成功能
- 集成深度学习算法进行情感分类
- 提供完整源代码及文档说明，便于理解与应用
